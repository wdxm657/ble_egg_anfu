- 3-设备定义
    - 蛋安抚器
        - 主要电子器件
            - 主控
                - RV1103G1，17RMB
                - 部署声音识别算法
            - 麦克风咪头
                - B6050AL-C1033，1.5RMB
            - 超声波模组
                - 双头+板，13.2RMB
            - 扬声器喇叭
                - 首选
                    - 高保真，自带音腔，6.2RMB
                - 次选
                    - 旧样机在用，1.1RMBf
            - 蓝牙芯片
                - TLSR8258，2RMB
        - 安装要求
            - 摆放在狗狗碰不到的地方，距离地面高度0.7~1.5米，正面无遮挡
        - 功能定义
            - 自动识别吠叫
                - 7 x 24 小时自动识别狗吠，监测到狗狗持续吠叫则自动执行安抚
                - 每10~15秒为一个监测周期，根据期间识别到的狗吠次数判断是否要执行安抚
            - 自动执行安抚
                - 无安抚成功记录时，按照出厂顺序执行安抚，直到狗吠停止或全部措施执行完
                    - 1-播放安抚音乐2分钟
                    - 如果吠叫未停止
                    - 已录制主人声音
                    - 2-播放主人录音
                        - 录音播放次数=ceil(20 / 录音长度)
                    - 如果吠叫未停止
                    - 3-释放超声波5秒
                        - 25kHz@110dB
                    - 如果吠叫未停止
                    - 4-释放超声波5秒
                        - 30kHz@110dB
                    - 如果吠叫未停止
                    - 5-释放超声波5秒
                        - 25kHz&30kHz@110dB
                    - 设备进入60秒休息时间
                - 如果有成功安抚记录，优先执行最近成功的安抚措施
                    - 根据成功安抚记录，基于出厂顺序，调整下次执行顺序
                    - 举例
                        - 最近成功止吠措施
                            - 25kHz@110dB
                        - 下次执行顺序
                            - 1-25kHz@110dB
                            - 2-30kHz@110dB
                            - 3-25kHz&30kHz@110dB
                            - 4-播放安抚音乐
                            - 5-播放主人录音
            - 主人录音管理
                - 录音长度：3~10秒
                - 可在APP控制设备进行
                    - 录音试听
                    - 录音清除
                    - 声音录制
            - 安抚措施可配置
                - 安抚措施可多选，至少选1个
                    - 播放安抚音乐
                    - 播放主人声音
                    - 发射超声波
                        - 发射频率及强度可多选，至少选1个，最多选3个
                            - 25kHz@100dB
                            - 30kHz@80dB
                            - 25kHz&30kHz@100dB
            - 安抚记录查看
                - 可在APP上查看最近30天的安抚记录
            - 恢复出厂
                - 在app上解除配对时，设备恢复出厂状态
                    - 清除主人录音
                    - 安抚措施执行顺序恢复出厂状态
                    - 断开与当前用户的蓝牙连接
            - 上电后自动开机，可通过app操作关机
        - 数据处理
            - 与APP的数据接口
                - APP请求设备
                    - 配对请求
                    - 解除配对请求
                    - 获取设备状态
                        - 开关状态

                    - 播放主人录音指令
                    - 删除主人录音指令

                    - 结束录制指令

                    - 音量，请求+指令
                    - 安抚模式，请求+指令
                    - 安抚措施，请求+指令
                - 设备主打上报
                    - 当前工作状态：监测/休息/识别/执行
                    - 安抚记录
                        - 吠叫识别时间
                        - 安抚措施执行时间（可能多个）
                        - 安抚结果及时间
        - 物理UI
            - 指示灯
                - 工作状态灯
                    - 熄灭：未供电
                    - 白色灯常亮：已供电、未开机
                    - 白色灯闪烁：开机启动中

                    - 选型
                        - 自动调整模式/Auto-Adjust Mode

                    - 说明文字
                        - 中文
                            - 自动调整模式
                                - 选择此模式，程序会自动根据安抚记录，优先执行最近有效的安抚措施
                            - 人工干预模式
                                - 选择此模式，程序会根据用户设置按照固定顺序执行安抚措施
                            - 通用规则
                                - 每执行一个安抚措施后，AI会识别狗吠是否停止，如果狗吠未停止，则执行其他未执行且可执行的措施，整个过程将持续至
                                    - 全部可执行措施均被执行
                                    - 狗吠已停止
                                    - 结束时上报执行过程和执行结果
                        - 英文
                            - Auto-Adjust Mode
                                - By selecting this mode, the program will automatically prioritize and execute the most recent valid soothing measures based on the soothing records.
                            - Manual Override Mode
                                - By selecting this mode, the program will perform calming measures in the order you set.
                            - General Rules
                                - After each soothing measure is executed, the AI will assess whether the barking has ceased. If barking persists, the system will proceed to implement the next available and unexecuted measure from the protocol list. This process continues until either:
                                    - Barking stops
                                    - All executable measures have been deployed
                                    - at which point the complete execution history and results will be uploaded

                                - 选择规则

                            - 英文
                                - Sequential Soothing Execution
                                    - When the device detects dog barking, it will execute selected soothing methods in a strict top-to-bottom sequence as configured in the app interface.
                                - Owner Voice Soothing Availability
                                    - The "Owner Voice Soothing" option will only be available if the owner's audio recording has been successfully captured and stored in the system.
                                - Selection rule
                                    - At least one calming measure must be selected.
                        - 选项
